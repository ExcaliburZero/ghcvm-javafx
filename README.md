# JavaFX in GHCVM

This is an example application that demonstrates Java integration in GHCVM

# Prerequisites
- JDK 8
- [GHCVM](https://github.com/rahulmutt/ghcvm) (On `master` branch)

# Running
```
$ cd ghcvm-javafx
$ cabalvm run
```
